Class arr

A collection of utilities for Arrays.

To get started, place the following line around top of your code.

Treats arguments as an one-dimensional array

Treats arguments as an one-dimensional array

Parameters

$X
Any number of parameters are accepted

Returns

Example

Converting a multi-dimentional array into one-dimentional

$var = array (
  'A',
  array (
    'B',
    array (
      'C'
    )
  ),
  'D'
);
$r = arr::flat($var); // $r = array ('A', 'B', 'C', 'D')

Converting multiple arguments into an one-dimentional array

$r = arr::flat('A', array ('B', 'C'), 'D');
    // $r = array ('A', 'B', 'C', 'D')
